# Cambios fiscales pendientes (catch file)

Drift regulatorio capturado durante el uso. **Ships vacío** salvo el drift ya conocido a la fecha de calibración. Cuando un usuario reporta o se detecta un cambio (nueva tabla ARCA, RG nueva, cambio cambiario, fallo), se anota acá; al confirmarlo contra la norma, se promueve al `reference/` correspondiente.

## Formato

`- **[fecha] [tema]** — qué cambió · fuente · reference a actualizar · estado (pendiente / confirmado / promovido)`

## Drift conocido a la calibración (jun-2026)

- **[~15-jul-2026] Tabla monotributo Ago2026–Ene2027** — ARCA publica la nueva tabla (+14–17% estimado, NO oficial aún) y la escala de Ganancias jul–dic 2026. Pendiente: actualizar `categorias-monotributo.md` y `monotributo-vs-ri.md` cuando salga. ESTADO: pendiente (esperado).
